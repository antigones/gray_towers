# gray_towers
Towers of Hanoi/Bucharest, the "Gray code way".

Example usage:


    from gray_hanoi import GrayHanoi


    def main():

        gray_hanoi = GrayHanoi(n_disks=3)
        gray_hanoi.solve(verbose=True)
